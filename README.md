# Arduino-Semáforo
Descrição:
Circuito arduíno feito com o objetivo de aprendizagem prática do sistema arduíno e aplicação dos conhecimentos de Algoritmo e Estrutura de Dados aprendidos até aquele ponto. O projeto simula um semáforo, utilizando sua lógica para se aproximar o máximo possível do que realmente acontece junto de uma função de troca repentina do ciclo após solicitação.

Tecnologias:
Para o projeto, foi utilizado um Arduíno UNO R3 e uma protoboard pequena como principais componentes, além de outros componentes para simular o semáforo. O código utilizado para fazer o circuito funcionar foi escrito na linguagem C.

Funcionalidades:
O projeto em sí não tem uma funcionalidade prática e aplicável, tendo em vista que ele foi desenvolvido como uma prática avaliativa para ensinar na prática a implementação do hardware com software.

Contexto:
O projeto foi criado como prática avaliativa dentro da disciplina de Trabalho Interdisciplinar 1 do 1° periódo do curso de Ciências da Computação.
